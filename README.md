# Projeto Acelera

Este projeto automatiza a distribuição de leads via webhook a partir de planilhas Google Sheets.

## Estrutura do Projeto

- `src/`: Código fonte Python
- `config/`: Arquivos de configuração (credentials.json)
- `logs/`: Arquivos de log de execução
- `assets/`: Recursos estáticos

## Instalação

1. Instale as dependências:
   pip install -r requirements.txt

2. Configure as credenciais em config/credentials.json

## Uso

Execute o script principal:
python src/Distribuidor_ACELERA.py
